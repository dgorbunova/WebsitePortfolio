Projects

IRIS

I created a personal assistant program using Python. The program was designed to help users with their daily tasks, such as setting reminders, creating to-do lists, and providing information on the weather. As well as this, it was voice acitvated and even had a wake word, it had capabilities to scrape a search engine and output the results directly to the user in the application. To further improve the application I added language functionality, at the moment it supports English, Spanish, French and German, this includes typing in the chat and spech recognition. The program is user-friendly and easy to use, with a simple and intuitive interface. The project involved a lot of problem-solving and critical thinking, this was especially the case for working with new libraries I had no experience with before, as well as a lot of programming and debugging. I learned a lot from this project, and it was a great experience to work on something that I was passionate about.
